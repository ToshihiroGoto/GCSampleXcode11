### macOS キーボード入力時のキー設定

|入力キー|機能|
|---|---|
|Z|攻撃|
|X|高速移動|
|C|立方体設置|
|カーソルキー|移動|
|A, D|カメラ左右回転|
|W, S|カメラ上下回転|
|スペースキー|状態リセット|
